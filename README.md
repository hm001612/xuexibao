# 学习宝 · 小学全科教辅软件

> 人教版语文 · 北师大版数学 · 人教PEP版英语（新教材）
> 一年级 ~ 六年级 全册覆盖

---

## 功能概览

| 功能 | 说明 |
|------|------|
| 📖 预习模式 | 带提示的探索性问题、词语翻转卡片、预读策略、思考题 |
| 🔄 复习模式 | 掌握打勾清单、易错点提醒、词语默写自测、快速自测题、知识图谱 |
| ✏️ 测试模式 | 按课 / 单元 / 期中 / 期末 / 自定义五种范围，8种题型，倒计时答题 |
| 📊 成绩分析 | 分数环 + 雷达图 + 错题解析 + 弱项专训入口 |
| 🖨️ 打印试卷 | A4标准版式，含答案纸，一键调起浏览器打印 |
| 📚 错题本 | 自动归档错题，支持针对弱项再训练 |

## 快速启动

```bash
# 克隆仓库
git clone https://github.com/hm001612/xuexibao.git
cd xuexibao

# 安装依赖
npm install

# 本地运行
npm run dev
# 浏览器访问 http://localhost:5173
```

> Node.js 18+ 即可运行，无需后端服务器。

## 技术栈

- **框架**：React 18 + Vite 5（纯前端 SPA）
- **路由**：状态驱动路由，通过 `navigate()` 切换屏幕
- **样式**：CSS 变量体系，"暖纸书房"主题（米白底 + 褐色文字）
- **字体**：ZCOOL XiaoWei · Noto Sans SC · DM Serif Display
- **图表**：SVG 手写雷达图（无第三方图表库）
- **打印**：`window.open + document.write` 生成 A4 打印页

## 题库说明

题库按年级分级存储，三科共 18 个年级题组，每组 5~7 题，覆盖不同题型：

| 科目 | 题型 |
|------|------|
| 语文 | 拼音汉字 / 词语积累 / 句子练习 / 阅读理解 / 古诗词默写 / 写作练习 / 判断题 |
| 数学 | 计算题 / 填空题 / 判断题 / 选择题 / 应用题 / 几何图形 |
| 英语 | 单词拼写 / 选择题 / 完形填空 / 阅读理解 / 句子翻译 / 情景对话 |

测试时按当前所选年级优先抽题，不足时补充基础题库。

## 预习 vs 复习对比

| 区块 | 预习模式 📖 | 复习模式 🔄 |
|------|------------|------------|
| 目标 | 探索性问题 + 提示（先猜后验） | 掌握打勾清单（含易/中/难标签） |
| 词语 | 翻转卡片，点击看含义 | 默写行式，手动揭晓 |
| 专属 | — | 易错点提醒 / 快速自测 / 知识图谱 |
| 练习 | 预测题 / 质疑题（开放） | 回忆题 / 迁移题（有参考答案） |
| 笔记 | 记录疑问，课后补充 | 总结掌握情况，列出待巩固点 |

## 测试类型

| 类型 | 范围 | 建议时长 |
|------|------|----------|
| 按课测试 | 单课知识点 | 20 分钟 |
| 单元测试 | 整单元综合 | 40 分钟 |
| 期中测试 | 第 1–4 单元 | 90 分钟 |
| 期末测试 | 全册综合 | 120 分钟 |
| 自定义测试 | 按题型自选 | 30 分钟 |

## 推荐学习流程

```
预习（课前）→ 上课 → 复习（课后）→ 按课测试 → 查看弱项 → 专项训练
```

1. 主控台选择年级/学期
2. 进入科目，找到即将学习的课，点击「开始预习」
3. 带着预习中的疑问上课
4. 课后切换「复习模式」，完成掌握清单和词语默写
5. 点击「开始测试」选择范围和题型
6. 查看成绩分析，雷达图定位弱项，点击「专项训练」针对练习
7. 需要纸质作业时，在测试配置页选「生成试卷」后打印
